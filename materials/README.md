## How to Use This Organization

### For Each Learning Unit:
1. Create a subfolder with the unit name (LE_I, LE_II, LE_III, LE_IV)
2. Add PDFs or materials to the appropriate folder
3. Create note files alongside materials
4. Reference materials when reviewing

### For Study Sessions:
1. Refer to `study_schedule.md` to see what's due each week
2. Find materials in the appropriate LE folder
3. Take notes and save progress
4. Check off items in `learning-progress.md`

### For Exercise Blocks:
1. Find exercise instructions in the Exercises/ folder
2. Work through the exercises
3. Save your solutions
4. Update your progress tracker

### For Thesis Preparation:
1. Review materials in Supplementary/Thesis_Resources/
2. Study the ExposÃ© exercise (Ãœbung X)

## Tips for Effective Organization

1. **Use consistent naming**: Include author names and topics
2. **Create index files**: LE I, LE II, III, IV each have their own README
3. **Keep related materials together**: Don't scatter one article across multiple folders
4. **Version control**: If you have multiple drafts of notes, use meaningful names
5. **Link across folders**: Your notes might reference materials from multiple LEs
6. **Regular updates**: Update the progress tracker as you work
